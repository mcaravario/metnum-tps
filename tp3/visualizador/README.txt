Es necesario tener instalado Python 2.7
https://www.python.org/download/releases/2.7

Para correr el visualizador se debe ejecutar:

python main.py <path_instancia> <path_tiro> <path_jugadores>

Por ejemplo:
python main.py instancias/rectas/r1.arq instancias/rectas/r1.tiro jugadores/vacio.out

Desde ya, cualquier duda o bug que encuentren,
nos escriben a metnum-doc@dc.uba.ar
